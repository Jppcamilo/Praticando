Treinamentos para serem praticados.

Página Pessoal Simples (HTML + CSS)

Descrição treinamento 1

Criar uma página com seu nome, uma foto (pode ser placeholder), uma bio curta e seus links de redes sociais.

Requisitos

  • Estrutura HTML semântica.
  • Estilização básica com CSS (cores, fontes, espaçamento).
  • Responsividade simples (mobile e desktop).

Card de Produto (HTML + CSS Flexbox)

Descrição treinamento 2

Criar um card para um produto com:

  • Imagem
  • Nome do produto
  • Preço
  • Botão "Comprar"

Bônus (se eu conseguir):

Ao passar o mouse, mudar a cor de fundo ou aplicar uma animação sutil no botão

Layout Responsivo com Grid

Descrição treinamento 3

Criar uma galeria de imagens com CSS Grid.

Requisitos:

  • De 4 colunas no desktop
  • 2 no tablet
  • 1 no celular
  • Usar @media queries

To-Do List com JavaScript

Descrição treinamento 4

Criar uma lista de tarefas onde o usuário pode:

  • Adicionar tarefas
  • Marcar como concluída
  • Remover tarefas

Página de Login com Validação

Descrição treinamento 5

Criar um formulário de login com validação em JavaScript.

Requisitos:

  • Campos: e-mail e senha
  • Validação ao clicar em “Entrar”:
  • E-mail válido (@ e .com)
  • Senha com pelo menos 6 caracteres
  • Mostrar mensagens de erro abaixo dos campos
